## 1. Database Schema Definition
**File:** `convex/schema.ts`

Define a strict data schema with appropriate validation rules and optimized lookup paths.

*   **Imports:** Utilize `defineSchema` and `defineTable` from `convex/server`, and type validators `v` from `convex/values`.
*   **Table Name:** `savedMeals`
*   **Fields & Validation:**

    *   `mealId`: `v.string()`
    *   `name`: `v.string()`
    *   `category`: `v.string()`
    *   `area`: `v.string()`

*   **Index Configuration:** Define the following indexes to optimize query paths:
    *   `by_user`: indexed on `["userId"]`
    *   `by_userMeal`: indexed on `["userId", "mealId"]`
